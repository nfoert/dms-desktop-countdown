# DMS Desktop Countdown

![Image showcasing the project, with the settings menu to the left and two widgets displaying countdowns on the right](/repo/images/image.png)

A simple plugin for Dank Material Shell that lets you create desktop widget countdowns.

- Count the days until an end date
- Provide a start date to get progress from then until the end date
- Display the hours, days, and weeks until the end date
- Only count certain days of the week, like how many mondays until the end date

## Usage:
1. Go to `Settings > Workspaces & Widgets > Desktop Widgets` and add a new `DMS Desktop Countdown` widget
2. Set the `Start Date` and `End Date` using the format `YYYY-MM-DD hh:mm:ss`
3. Set `Show hours`, `Show days`, and `Show weeks` as desired
4. Use `Count <Weekday>` to limit the counts to only be on those days
